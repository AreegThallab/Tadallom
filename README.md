# Tadallom (تظلُّم)

منصة **Tadallom** هي نموذج أولي (MVP) يساعد المستخدم على **تنظيم بيانات التظلّم** وتجهيز **تقرير داعم يثبت التظلّم** يمكن استخدامه كدليل عند رفع الدعوى أو تقديمه للجهة المختصة.

> ملاحظة: المنصة تُسهّل تنظيم المعلومات وإخراج تقرير داعم، ولا تُعتبر استشارة قانونية.

---

## الفكرة
المستخدم يكتب تفاصيل القرار الإداري/المشكلة وما يطالب به، ثم تقوم المنصة بـ:
- ترتيب البيانات وتنسيقها بشكل واضح
- التحقق من اكتمال المعلومات الأساسية
- **حساب المدة الزمنية/مهلة التظلّم** بالاعتماد على النموذج المستخدم داخل النظام
- إصدار **تقرير داعم** قابل للحفظ/الطباعة والاستخدام كمرجع

---

## أبرز المزايا
- إدخال بيانات التظلّم والقرار الإداري
- تتبع حالة التظلّم (مثل: تم الإرسال / بانتظار رد / لم يتم الرد)
- إنشاء تقرير يدعم التظلّم ويجمع المعلومات بشكل منظم
- حساب المدة الزمنية/مهلة التظلّم للمستخدم
- واجهة بسيطة وسهلة الاستخدام

---

## التقنيات المستخدمة
- PHP + MySQL
- HTML / CSS / JavaScript
- Python (Model) لحساب المدة الزمنية/مهلة التظلّم
